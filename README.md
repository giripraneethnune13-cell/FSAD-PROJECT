# Student Project Portfolio Platform

A full-stack web application for students to manage and showcase projects.

## Stack
- **Frontend**: React + Vite (Glassmorphism UI)
- **Backend**: Spring Boot 3 + Java 17
- **Database**: MySQL

## Run Locally

### Backend
```bash
cd backend
mvn spring-boot:run
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## Features
- JWT Authentication (Student / Admin roles)
- Project CRUD with milestone tracking
- Media uploads: Images, PDFs, Videos
- Admin feedback & ratings
- Public portfolio pages
